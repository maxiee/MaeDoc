## 格式说明

每条待办事项遵循以下格式：

```markdown
### TODO-{编号}: {标题}

- **来源**：{哪个 Command/Skill 创建的}
- **创建时间**：{YYYY-MM-DD HH:MM}
- **优先级**：P0 / P1 / P2
- **相关文档**：{文件路径，可选}
- **描述**：
  {具体问题描述或待完成事项}
```

**优先级定义**：

| 级别 | 含义 | 示例 |
|------|------|------|
| P0 | 阻塞性问题，需立即处理 | 文档结构严重不一致 |
| P1 | 重要但不紧急，近期处理 | 补充缺失章节内容 |
| P2 | 低优先级，有空时处理 | 优化文档措辞 |
