## 记录规范

每条晶体必须包含：

1. `Claim`：可迁移的论断
2. `Confidence`：`[HIGH]` / `[MED]` / `[LOW]`
3. `Evidence`：至少 1 个文档证据链接
4. `Apply Scope`：适用边界
5. `Counter Evidence`：反证或不成立条件
